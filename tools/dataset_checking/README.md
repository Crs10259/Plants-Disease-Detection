# Data Checking Tool

This tool provides a graphical interface for manually checking and verifying image datasets.

## Features
- Single image review mode
- Multiple images review mode
- One-click delete functionality
- User-friendly PyQt6 interface

## Installation
1. Run `install.bat` to set up the required dependencies
2. Run `run_tool.bat` to start the application

## Requirements
- Python 3.8+
- PyQt6
- Pillow

## Usage
1. Launch the tool using `run_tool.bat`
2. Select the directory containing images to check
3. Use the interface to review and manage images:
   - Single image mode: Review one image at a time
   - Multi image mode: Review multiple images simultaneously
   - Use delete button to remove unwanted images
